About The Project

"Tranquility" is an original R&B composition created as part of a university project. While the project began with inspiration and feedback from two fellow students, the final track evolved into a completely independent and personal production.

Learn more

Production Process

The track was composed, arranged and mixed from scratch using digital production tools through FL studio. The goal was to create a calm and emotional atmosphere through smooth harmonies, ambient textures and modern R&B drums.

Learn more

Musical Inspiration

The sound of "Tranquility" draws inspiration from modern alternative R&B, focusing on emotional depth, atmospheric melodies and minimal production.
